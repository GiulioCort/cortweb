Elementi

Un elemento HTML è definito da:

  • Un tag iniziale
  • Del contenuto
  • Un tag finale

Degli esempi di elementi HTML: <h1>Intestazione</h1>
Alcuni elementi non hanno né contenuto né tag finale.

Elementi HTML annidati

Gli elementi HTML possono essere annidati, cioè si può fare in modo che alcuni elementi siano contenuti in altri elementi.
Per esempio, tutti gli elementi contenuti nell’elemento <body>, ad esempio l’elemento <p>, sono annidati dentro il body.

Tag finale

Alcuni elementi non richiedono un tag finale, ad esempio l'elemento <p>

Esempio

<!DOCTYPE html>
<html>
<body>

<p>Un paragrafo.
<p>Un altro paragrafo.

</body>
</html>
Risultato

Elementi HTML vuoti

Alcuni elementi HTML possono non avere un contenuto, ad esempio il tag <br>.

Case sensitive

HTML non è case sensitive, cioè i tag possono essere scritti sia con lettere minuscole che maiuscole. <p> è uguale a <P>.
