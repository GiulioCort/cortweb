Attributi

In HTML, gli attributi forniscono delle proprietà ad un elemento. Gli attributi sono specificati sempre nel tag iniziale di un documento. Gli attributi HTML hanno sempre questa struttura: attributo="valore".

Ecco alcuni attributi, anche se alcuni li vedremo con più dettaglio nei capitoli successivi.

L'attributo href

Il tag <a> definisce un link. L'attributo href specifica l’URL della pagina a cui il link ti porta.

Esempio

<p>I link sono definiti dal tag a. L'indirizzo del link è specificato dall'attributo href.<p>
<a href="https://giuliocort.github.io/cortweb/">Visita il sito.<a>
Risultato

L'attributo src

Il tag <img> è usato per inserire un'immagine in una pagina HTML. L'attributo src specifica la posizione dell'immagine inserita.

Esempio

<p>Un'immagine su HTML viene definita con il tag img. L'attributo src serve a specificare la posizione in cui si trova l'immagine.<p>
<a src="Rhod.jpg" width="700" height="500">
Risultato
