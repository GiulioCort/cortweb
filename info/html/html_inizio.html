Introduzione

HTML (HyperText Markup Language) è il linguaggio di marcatura più usato per i documenti web. Un linguaggio di markup è un linguaggio che consente di descrivere dati tramite marcatori, detti tag.

Istruzioni

Per iniziare a creare pagine web occorre semplicemente disporre di un editor di testo, come il blocco note di windows. Nonostante sia possibile scrivere un file HTML con il semplice blocco note, consiglio di utilizzare un programma più adatto tipo Notepad++ o Visual Studio Core.
Creare un file HTML è semplice: basta creare un file e rinominarlo con l’estensione .html. ES. nomefile.html.
Il file HTML si deve poi aprire con l’editor di codice scelto. Una volta scritto il codice basta salvarlo e aprirlo nel browser, dove viene visualizzato sotto forma di pagina web.

Basi

Il codice HTML è composto da una serie di elementi, detti tag, che sono annidati l’uno dentro l’altro. Ogni pagina web è costituita da una struttura di base.

Esempio

<!DOCTYPE html>
<html>
<head>
<title>Titolo della pagina</title>
</head>
<body>

<h1>Intestazione</h1>
<p>Paragrafo.</p>

</body>
</html>
Risultato

Ora è meglio spiegare la struttura sovrastante:

  • Il tag !DOCTYPE html serve a stabilire che il documento è un documento HTML5.
  • Il tag html è fondamentale in ogni documento HTML. Sancisce l’inizio e la fine di una pagina HTML.
  • Il tag head contiene informazioni sulla pagina HTML.
  • Il tag title serve a specificare il titolo della pagina HTML, mostrato nella barra del browser.
  • Il tag body definisce il corpo della pagina, l’equivalente del testo di un documento word. Questo tag contiene tutti i contenuti visibili nella pagina, come testi, immagini, tabelle, ecc.
  • Il tag h1 definisce un intestazione.
  • Il tag p definisce un paragrafo di testo.

Un elemento HTML è composto da un tag iniziale, del contenuto e un tag finale. Un esempio è il tag h1 dove, tra i due tag, si trova il contenuto che si vuole andare a inserire. È importante ricordarsi che i tag finali devono iniziare con /.
Alcuni tag, come il tag br, che vedremo in seguito, non necessitano di un contenuto o di un tag finale.
